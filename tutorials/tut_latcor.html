The following example uses structrans.lat_cor() to find the best two lattice correspondences

  • from face centered cubic with a = 2
  • to primitive tetragonal with a = 1.414 and c = 2.

The Bravais lattice id and associated lattice parameter lists can be found in the documentation of here.

>>> from structrans import lat_cor
>>> res = lat_cor(2, 2, 6, [1.414, 2], disp=2, nsol=2)

==========

    3D Bravais Lattice - Cubic F (face centered cubic):    a = 2
 - Martensite lattice:
