Lattice¶

The class Lattice handles the generic lattice objects.

Definition

A lattice \(\mathcal L\) is a set of discrete points in \(\mathbb R^N\). It is defined through three linearly independent lattice vectors, \(\mathbf e_i\in\mathbb R^N\), \(i = 1, \cdots, N\):

(1)\[\mathcal{L}(\mathbf e_i) = \{\mathbf x\in\mathbb R^N : \mathbf x = \mu^i\mathbf e_i, \mu^i \in \mathbb Z\}.\]

In terms of matrix multiplication, each point in \(\mathcal L\) is obtained by mutltiplying the base matrix \(\mathbf E\in\mathbb R^{N\times N}\) by an integer vector \(\boldsymbol\mu\in\mathbb Z^N\), i.e.

\[\begin{split}\mathbf x = \mu^1\mathbf e_1 + \cdots + \mu^N\mathbf e_N = \begin{bmatrix} e^1_1 & \cdots & e^1_N \\ \vdots & \ddots & \vdots \\ e^N_1 & \cdots & e^N_N \end{bmatrix} \begin{bmatrix} \mu^1 \\ \vdots \\ \mu^N \end{bmatrix} = \mathbf E\boldsymbol \mu.\end{split}\]

Then the definition (1) is equivalent to

\[\mathcal{L}(\mathbf E) = \{\mathbf x\in\mathbb R^N : \mathbf x = \mathbf E\boldsymbol\mu, \boldsymbol\mu \in \mathbb Z^N\}.\]

This definition is valid for any \(\mathbf E\in\mathbb R^{N\times N}\), such that \(\text{det }\mathbf E\neq0\).

Equivalence

Two lattice are equivalent if and only if their bases are related through a unimodular integer matrix in the general linear group \(GL(3, \mathbb Z)\):, i.e.

\[\mathcal L(\mathbf E_1) = \mathcal L(\mathbf E_2) \iff \mathbf E_1\mathbf L = \mathbf E_2, \mathbf L \in~GL(3, \mathbb Z)\]

Lattice object¶

class structrans.Lattice(E)¶

This is the base class of lattices.

Attribute E:two-dimensional square matrix (any type can be converted to numpy.ndarray)
Laue_group()¶
Returns:get the Laue group in \(SO(3)\) of a 3D lattice, in \(SO(2)\) of a 2D lattice.
Return type:MatrixGroup
Raises AttributeError:
 Laue groups for other than 2D and 3D lattices are not implemented yet.
base()¶
Returns:the base matrix
Return type:numpy.ndarray
dimension()¶

Get the dimenstion of the lattice.

Returns:(integer) - the dimenstion of the lattice
inlatticegroup(M)¶

Check if M is in the lattice group of the lattice. This method works for lattices in any dimension.

inpointgroup(Q)¶

Check if Q is in the point group of the lattice. This method works for lattices in any dimension.

Examples¶

>>> from structrans import *
>>> import numpy as np
>>> lat = Lattice([[1., 1., 0],[-1., 1., 0],[0, 0, 1.]])
>>> E = lat.base()
>>> print(E)
[[ 1.  1.  0.]
 [-1.  1.  0.]
 [ 0.  0.  1.]]
>>> print(lat.dimension())
3
>>> print(lat)
3 dimensional lattice:
    e_1 = [ 1. -1.  0.]
    e_2 = [ 1.  1.  0.]
    e_3 = [ 0.  0.  1.]

Note

Lattice is an immutable object. Once instantiated, the base matrix \(\mathbf E\) cannot be changed.

>>> E[0, 1] = 0
>>> print(E)
[[ 1.  0.  0.]
 [-1.  1.  0.]
 [ 0.  0.  1.]]
>>> print(lat.base())
[[ 1.  1.  0.]
 [-1.  1.  0.]
 [ 0.  0.  1.]]

>>> L = np.array([[1, 1, 0], [0, 1, 0], [0, 0, 1]])
>>> import numpy.linalg as la
>>> la.det(L)
1.0

>>> E2 = lat.getbase().dot(L)
>>> lat2 = Lattice(E2)
>>> lat == lat2
True
